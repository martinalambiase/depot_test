Les Anneaux

1Trois Anneaux pour les Rois Elfes sous le ciel,
2Sept pour les Seigneurs Nains dans leurs demeures de pierre,
3Neuf pour les Hommes Mortels destinés au trépas,
4Un pour le Seigneur Ténébreux sur son sombre trône,
5Dans le Pays de Mordor où s'étendent les Ombres.
6Un Anneau pour les gouverner tous. Un Anneau pour les trouver,
7Un Anneau pour les amener tous et dans les ténèbres les lier
8Au Pays de Mordor où s'étendent les Ombres.
